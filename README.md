# 🧮 Calculator App

A simple and responsive Calculator application built using HTML, CSS, and JavaScript. This project performs basic arithmetic operations with a clean and modern user interface.

---

## 📸 Preview

![Calculator UI](calculator.png)

---

## ✨ Features

- Clean and modern UI
- Responsive calculator layout
- Basic arithmetic operations
  - Addition (+)
  - Subtraction (-)
  - Multiplication (×)
  - Division (÷)
  - Percentage (%)
- Clear (CL) button
- Delete (DEL) button
- Real-time display updates
- Easy to understand and modify

## 📱 User Interface

The calculator includes:

- Display Screen
- Number Buttons (0–9)
- Arithmetic Operators
- Percentage
- Delete Button
- Clear Button
- Equal Button

---

## ⚙️ How It Works

1. Click the number buttons to enter values.
2. Select an arithmetic operator.
3. Enter the second number.
4. Press **=** to calculate the result.
5. Use **DEL** to remove the last digit.
6. Use **CL** to clear the display.
